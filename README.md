# zhm-leyusports-guide

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或具体网站，仅作为静态页面的集合与展示。

## 项目简介

本项目旨在将一系列独立的 HTML 页面集中存放，便于版本管理与访问。页面内容可能涵盖说明、指南、工具界面或其他信息性内容，彼此之间无强关联。每个页面均为独立文件，可单独查看。

## 目录说明

仓库根目录下直接放置 HTML 文件，部分页面可能引用同目录下的资源（如 CSS、JS 或图片）。建议通过文件列表浏览所有可用页面。

```
zhm-leyusports-guide/
├── index.html          # 入口页面（如有）
├── page-a.html         # 示例页面 A
├── page-b.html         # 示例页面 B
└── assets/             # 公共资源目录（可选）
    ├── style.css
    └── script.js
```

*注：实际文件以仓库内容为准。*

## 页面归档说明

- 每个 HTML 页面均为独立、完整的文档，可直接在浏览器中打开。
- 页面间无跨域依赖，所有资源引用均为相对路径或内联。
- 部分页面可能包含纯前端交互逻辑，不依赖后端服务。
- 页面内容不保证与任何外部网站或服务有关联。
